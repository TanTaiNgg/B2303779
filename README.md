## 📋 Mô Tả Đồ Án

Đây là một **website thương mại điện tử (E-Commerce)** chuyên bán đồ ăn chay và các sản phẩm liên quan đến lối sống lành mạnh.

### ✨ Tính Năng Chính

- 🛒 **Giỏ hàng** - Quản lý sản phẩm và thanh toán
- 🔐 **Xác thực người dùng** - Đăng ký, đăng nhập, quản lý tài khoản
- 📦 **Quản lý sản phẩm** - Danh sách đồ ăn chay đa dạng
- 💳 **Thanh toán** - Hỗ trợ nhiều phương thức thanh toán
- 📱 **Responsive Design** - Tương thích với mọi thiết bị
- ⭐ **Đánh giá sản phẩm** - Hệ thống nhận xét và rating
- 📍 **Giao hàng** - Theo dõi đơn hàng

## 🚀 Hướng Dẫn Cài Đặt

### Yêu Cầu Hệ Thống
- Browser hiện đại (Chrome, Firefox, Edge, Safari)
- Server web (Apache, Nginx, Node.js)

### Các Bước Cài Đặt

1. **Clone repository**
   ```bash
   git clone https://github.com/TanTaiNgg/B2303779.git
   cd B2303779
   ```

2. **Cài đặt dependencies** *(nếu có)*
   ```bash
   npm install
   ```

3. **Chạy server**
   ```bash
   npm start
   # hoặc
   php -S localhost:8000
   ```

4. **Truy cập ứng dụng**
   ```
   http://localhost:8000
   ```
